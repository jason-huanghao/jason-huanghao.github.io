Research Pipeline

Posted on 2021-07-13 Edited on 2021-07-15 In Research Method Views: Views: Valine: 2.9k 3 mins.

Things become easy if you arrange it into industrial production procedure. Publishing a paper can also be treated like a pipeline work procedure.

Paper Searching

  1. ACL Anthology for ACL top NLP conference paper free download! ❤️
  2. Connected Papers to find papers that connect with the query paper. The results will be shown in a graph, awesomoe!
  3. Semantic Scholar search paper by different filters (fileds of study, date, has pdf, publication type, famous authors, journal & conference), could be a super search tool
  4. CiteSpace, HistCite Pro (both are software) are used for analyze the citation network
  5. Library Genesis facilitate you to download every pdf book
  6. 大木虫学术导航 is a comprehensive nevigation for scholars, 有空多逛逛
  7. 虫部落 very good collection of search tools and methods

Paper Download

  1. ACL Anthology ❤️
  2. 虫部落学术搜索板块
  3. Google Search with filetype:pdf

Note Taking

  1. Hexo-Next A self deployed blog for personal notes and blogs
  2. TiddlyWiki A personal knowledge system, also a unique non-linear notebook for capturing, organising and sharing complex information. They have a TiddlyWiki Forum for problem solving. There is the TyddlyWiki5 Github Repository and a Desktop Tool.
  3. Typora (software) The best markdown edit tool
  4. VScode (software) The best editor for code and for multiple use, many plugins

Coding

  1. Source Graph is a tool for searching code through all github repositories
  2. Paper with Code is a community where every work will be attached by their code link
  3. Jupyter Notebook, Jupyter Lab, Pycharm (software) for python writing

Paper Writing

  1. Environment:

    • Focus Music for writing
    • Quite Place with coffee or tea

  2. Writing Tool

    • Overleaf with latex colaborate with others by using latex

  3. English Writing

    • Thesaurus.com enrich your lexical resource
    • Academic Word List most frequent words used in academic paper
    • Academic Phrasebank help you to polish your sentences so that equip your paper with coherence and logic.
    • Linggle.com help you to chose right words for your expression.
    • Grammarly check your spell errors and grammar errors.
    • DeepL for translation

  4. Formating

    Color

    • Colors.co for color choosing

    Latex Formula

    • Snip for hand writing formula and table recognization. I would highly recommand for generating the latex code of complex table, which is quite convinient.

    Figures

    • Draw.io is the best tool to draw a framework or procedure
    • PlotNeuralNet or for plotting neural network
    • Omnigraffle (software on MacOS or PadOS) a excellent tool for drawing
    • TikZ supports latex for drawing
    • Keynote or PPT (software) is also good

Journal or Conference

Publication Ranking

Journals - SJR helps you to choose a suitable Journal in your area. Ranking Journal - LetPub is also a good platform ranking Journal

Conference - Conference Ranks for conferences - CORE Computer Science Conference Rankings helps you to estimate the rank of conferences

Traking Deadlines

  • AI Conference Deadlines informs and tracks the deadlines of all AI conferences

Presentation

  • Keynote
